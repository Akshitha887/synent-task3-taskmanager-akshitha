# Task Manager Application

## Overview

A Full Stack Task Manager built using Flask and SQLite.

## Features

* Add Tasks
* View Tasks
* Store Tasks in SQLite Database
* Flask Web Application
* HTML Templates

## Technologies Used

* Python
* Flask
* SQLite
* HTML

## Project Structure

task-manager/
│
├── app.py
├── database.py
├── templates/
│ ├── index.html
│ └── about.html

## Run Project

1. Install Flask

pip install flask

2. Create Database

python database.py

3. Run Application

python app.py

4. Open Browser

http://127.0.0.1:5000
